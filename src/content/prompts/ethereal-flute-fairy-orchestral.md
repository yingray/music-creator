## Notes

A fairy-tale orchestral piece led by ethereal flute. Uses chamber orchestra (not full symphony) to maintain lightness and transparency. The Impressionist-era harmonic language provides the right color palette — luminous but not heavy.

### Key Distinction from Ghibli Prompt
- Ghibli prompt builds from serene to epic/majestic; this stays delicate throughout
- This uses "chamber orchestra" (lighter); Ghibli uses full orchestral build
- "Fairy-like" and "secret realm" give a more fantastical character vs. Ghibli's spiritual/mythic quality
- Shared: both use transparent string writing and organic acoustic orchestration

### Key Descriptors
- `ethereal flute solo`, `arpeggiated harp`, `magical forest`, `fairy-like`
- `glissando textures`, `soft pizzicato`, `pastoral elegance`, `secret realm`
