## Notes

Gershwin-style symphonic jazz rhapsody — the genre that fused classical orchestral writing with jazz harmony and rhythm. The prompt deliberately balances both worlds: orchestral weight and formality with jazz's blue notes, syncopation, and swagger.

Reference point: *Rhapsody in Blue* (1924) — clarinet glissando opening, virtuosic piano, rhapsodic form, jazz harmonies within orchestral texture.

### Key Distinction from Rachmaninoff Prompts
- Rachmaninoff is "Russian late-Romantic, minor key, melancholic power" → this is "American Jazz Age, major-mode exuberance, urban vitality"
- Both feature virtuosic solo piano with orchestra, but the harmonic language is fundamentally different (chromatic Romantic vs. blue-note jazz)
- Rachmaninoff uses sonata form → this uses rhapsodic free-flowing structure

### Key Distinction from Debussy Prompt
- Debussy is "French Impressionist, soft dynamics, suspended" → this is "American, high-energy, rhythmically driven"
- Both are early 20th century but from opposite aesthetic poles

### Key Descriptors
- `symphonic jazz rhapsody`, `clarinet glissando opening`, `blue note harmonies`
- `syncopated rhythmic energy`, `swinging orchestral passages`, `stride piano flourishes`
- `1920s American Jazz Age spirit`, `rhapsodic free-flowing structure`

### Effective Patterns
- "Gershwin-inspired but fully original" — consistent with project's composer-anchor pattern
- "early 20th-century American symphonic idiom" — era + cultural anchor prevents both Romantic drift and modern jazz drift
- "jazz-inflected blue note harmonies" — specifies the jazz element is in harmony, not performance style
- "alternating between exuberant fast sections and tender lyrical episodes" — describes the rhapsodic form's contrast structure
- "blue thirds and sevenths" — precise music theory term for the jazz harmonic color
- "stride piano flourishes" — era-appropriate piano style (not bebop, not Romantic)

### Negative Prompt Strategy
Inherits the standard classical exclusion list plus additional jazz sub-genre exclusions (`bebop, free jazz, smooth jazz, fusion jazz`) to prevent the jazz element from drifting into wrong jazz era or style. The "jazz drums" exclusion is critical — we want jazz *harmony*, not jazz *rhythm section*.

### v5 Notes
- v5 handles the dual-genre anchoring ("symphonic jazz") well — it can hold both classical and jazz vocabularies simultaneously
- "clarinet glissando opening" is specific enough that v5 tends to produce a clarinet-forward intro
- The rhapsodic structure (no strict verse/chorus) works well with v5's instrumental mode
- "syncopated rhythmic energy" guides v5 to produce swing-inflected rhythm without needing a drum kit
