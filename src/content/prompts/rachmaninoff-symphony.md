## Notes

A Rachmaninoff-style symphony — distinct from the concerto set because there is **no solo instrument**. The orchestral texture must be self-sufficient, with dialogue happening between sections rather than between soloist and orchestra.

### Key Distinction from Concerto Set
- "no solo instrument" explicitly prevents Suno from introducing a piano soloist
- "symphonic fugato" and "complex polyphony" replace the concerto's "dialogue between piano and orchestra"
- "noble brass chorales" takes a more prominent role here since brass isn't supporting a soloist
- "non-quantized performance" from the original handwritten prompt — tells Suno to avoid mechanical rhythms

### Key Descriptors
- `absolute music`, `symphonic fugato`, `complex polyphony`, `non-quantized performance`
- `noble brass chorales`, `deep orchestral bass`, `sweeping melodic development`

### Effective Patterns
- "Rachmaninoff-inspired but fully original" — same anchoring pattern as the concerto set
- "absolute music" — prevents programmatic/narrative drift
- "no solo instrument" — critical anti-pattern anchor to prevent concerto-like textures
- "non-quantized performance" — encourages natural rubato across the whole orchestra

### v5 Notes
- v5's improved instrument separation helps differentiate the woodwind dialogues from the string section
- The comprehensive negative prompt (35 terms) shared with the concerto set remains necessary
- "no solo instrument" works well in v5 as an explicit exclusion within the positive prompt
